# Style Hard Constraints (LayerAxis V4 Flow)

## Mandatory

- Perspective: isometric 2.5D only.
- Visual language must stay in one domain for multi-panel images.
- Core objects per panel <= 3, action hint <= 1.
- Total identifiable visual units <= 7.
- All visible text in Chinese. No English text.

## Forbidden

- Flat infographic substitution when scene requires depth cues.
- Emoji in prompts.
- Neon/cyberpunk/high-saturation drift.
- Mixed perspective systems in one image.
