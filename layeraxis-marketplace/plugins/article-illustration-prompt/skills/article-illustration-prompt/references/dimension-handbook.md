# 正交维度参考手册

> 思路卡住时的备查参考。大多数时候，跟着原文意象走就够了。

---

## 隐喻类型速查

| 类型 | 物件举例 | 强调 |
|---|---|---|
| **容器** | 盒子、抽屉、档案柜、房间 | 存储、归类、内外边界 |
| **路径** | 道路、河流、轨道、阶梯 | 方向、进程、旅程 |
| **工具** | 望远镜、钥匙、滤网、杠杆、旋钮 | 能力、聚焦、转化、调节 |
| **生态** | 种子→树、土壤、花、天气 | 生长、衰败、条件、演化 |
| **空间** | 地图、坐标系、楼层、地形 | 定位、全局视野、维度 |

**选择优先级**: AI 渲染稳定性 > 信息传达力 > 艺术表现力

## 常见转译

| 抽象概念 | → 名词物件 | 物件状态描写 |
|---|---|---|
| 时间流逝 | 沙漏 | 玻璃壁微微变形，沙粒流动 |
| 衰败 | 枯萎的花 | 花瓣下垂卷曲，茎微弯 |
| 过载 | 溢出的文件柜 | 抽屉敞开，纸张散落 |
| 筛选 | 漏斗/滤网 | 上方杂乱，下方清晰 |
| 可分离 | 叠加层(间距增大) | 半透明，物理间距编码解耦性 |
| 封闭→开放 | 堆叠→扇形展开 | 从紧贴到角度散开 |
| 一维→多维 | 直线→L形坐标系 | 点序列变双轴分布 |
| 组合/汇聚 | 拼图/积木 | 多块咬合拼合 |
| 迭代 | 环形箭头 | 从末端弧形回指 |
| 不可归类 | 漂浮在外的卡片 | 悬浮、倾斜、虚线指向但以问号终止 |

## 情绪修辞手法

| 手法 | 视觉实现 |
|---|---|
| **规模感** | 极小人物 vs 巨大结构 |
| **压迫感** | 密集排列、暗色调 |
| **轻盈感** | 大面积留白、漂浮元素 |
| **秩序感** | 精确对齐、网格、统一间距 |
| **温度感** | 暖色调、有机形状 |
| **空茫感** | 背景渐暗、雾化、虚影 |
| **顿悟感** | 模糊→清晰的对比 |
