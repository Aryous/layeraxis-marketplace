# 正交维度参考手册

> 思路卡住时的备查参考。大多数时候，跟着原文意象和自然结构走就够了。

## 结构类型

对比/并置、流程/时间线、层级/分层、协作/交互、矩阵/象限、演变/迭代、辐射/关联、循环

## 隐喻类型

| 类型 | 典型物件 | 强调 |
|------|----------|------|
| **容器** | 盒子、抽屉、房间、大脑 | 存储、归类、内外边界 |
| **路径** | 道路、河流、轨道、阶梯 | 方向、进程、旅程 |
| **工具** | 望远镜、钥匙、滤网、杠杆 | 能力、聚焦、转化 |
| **生态** | 种子→树、土壤、天气、季节 | 生长、条件、自然演化 |
| **空间** | 地图、坐标系、楼层、地形 | 定位、全局视野、相对关系 |

## 情绪修辞

| 类型 | 视觉手法 |
|------|----------|
| **规模感** | 极小人物 vs 巨大结构，俯瞰视角 |
| **压迫感** | 密集排列、暗色调、逼近视角 |
| **轻盈感** | 大面积留白、柔和色调、漂浮元素 |
| **秩序感** | 精确对齐、网格布局、统一间距 |
| **温度感** | 暖色调、有机形状、手绘质感 |
