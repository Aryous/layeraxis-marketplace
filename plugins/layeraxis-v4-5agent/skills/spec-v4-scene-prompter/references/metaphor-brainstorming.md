# 隐喻 Brainstorming 流程

> 当某张图的隐喻包装评分 ≥ 70 时触发本流程。

## 分支判断

- **原文有具象意象** → 直接采纳，用下方三项标准快速检验即可
- **原文无具象意象** → 发散 3 个候选 → 快速择优 → 采纳

## 候选发散

针对目标概念，生成 3 个候选隐喻意象：

1. 每个用一句话描述：「什么物件 + 什么状态 + 传递什么感受」
2. 候选之间走不同隐喻类型（容器/路径/工具/生态/空间，详见 @references/dimension-catalog.md）
3. 优先选 AI 画得好的常见具象物件，避免抽象/动态场景

## 快速择优

用三项标准评估每个候选，标记 ✓（好）/ △（一般）/ ✗（差）：

| 标准 | 核心问题 | ✓ 的信号 | ✗ 的信号 |
|------|----------|----------|----------|
| **语义契合** | 读者能从画面联想到原文概念吗？ | 直觉关联，不需要解释 | 需要文字说明才能理解 |
| **AI 可渲染** | 全是名词？物件清晰？ | 常见具象物件，静态场景 | 抽象概念、复杂动态、多重隐喻 |
| **风格兼容** | 和当前风格指南的视觉语言匹配吗？ | 几何/理性/克制 | 卡通/写实/高饱和 |

选 ✓ 最多的候选。平局时按优先级：**AI 渲染稳定性 > 信息传达力 > 艺术表现力**。

附一句选择理由。

## 隐喻展开

> 你选了一个隐喻。但"坐标系"是一个类型标签，不是一份施工图纸。
> 现在把它展开成可以直接画的视觉规格。

选中隐喻后，逐项回答：

- **物件形态**：具体长什么样？什么状态？
  不是"字典"，是"翻开到中间、三页被整齐撕掉露出锯齿边缘的厚精装字典"
- **尺寸比例**：在画面中占多大？和其他元素什么比例关系？
- **画面位置**：居中？偏到一角？悬浮在上方？
- **标注文字**：物件上或旁边标什么中文？写出完整的标题 + 副标题
- **空间关系**：旁边有什么？前后/上下/内外怎么摆？
- **语义标记**：需要 ✓（可行）、✗（不可行）、?（未知）来辅助读者理解吗？

展开的结果直接写入场景设计的"逐面板/主体"部分。如果展开时发现隐喻写不出具体形态——说明选错了，回到候选列表换一个。

## 输出示例

```
- 候选1：翻译词典缺页 — 语义✓ 可渲染✓ 风格✓ → 选中
- 候选2：悬崖断桥 — 语义✓ 可渲染✓ 风格△
- 候选3：乱码终端 — 语义△ 可渲染✓ 风格✓
选择理由：词典的文化共鸣最直觉，且书籍是 AI 渲染最稳定的物件之一

展开：
- 物件形态：厚精装词典，深海军蓝封面，翻开到中间，三页被整齐撕掉露出锯齿边缘
- 尺寸：占画面中央约 60%，略向左倾斜 10°
- 位置：画面中央偏下，下方有接触阴影
- 标注：左页印 bold "原文语义"，右页空白处一个琥珀色 "?" 悬浮
- 空间关系：词典右侧散落两张被撕下的页面，页面上有模糊的文字
- 语义标记：琥珀色 ? 表示"翻译缺失"
```
