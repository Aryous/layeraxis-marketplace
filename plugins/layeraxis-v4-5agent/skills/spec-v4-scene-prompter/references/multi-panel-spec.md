# 多面板图规范

> 当一张图包含多个并列面板（三联画、对比图等）时，加载本参考。

## 画布

- 横向多面板：**16:9**，提示词首段声明 `wide horizontal composition (16:9)`
- 竖向多面板：**9:16**

## 容器

- 每个面板用**物理卡片**承载（白色背景、描边、圆角、阴影）
- 不用「面板」「区域」等模糊词，用 card
- 卡片规格在场景描述之前统一定义

## 视觉导航

- 每张卡片左上角加**编号**（"1" / "2" / "3"），深色粗体
- 编号下方加**中文副标题**

## 面板间连接

- 底部用**水平虚线**连接所有面板
- 虚线下方放置**总结性文字**，用强调色
- 避免粗箭头横贯底部

## 面板内约束

- 核心物件 ≤ 3 个，动作暗示 ≤ 1 个
- 所有面板使用同一视觉域的物件（禁止混搭）
